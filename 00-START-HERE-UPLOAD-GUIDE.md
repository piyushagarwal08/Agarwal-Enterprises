## ⚙️ Before Uploading

### Step 1: Change Password in index.html
1. Open `index.html` in text editor
2. Find line: `const ADMIN_PASSWORD = 'admin123';`
3. Change `'admin123'` to YOUR password
4. Example: `const ADMIN_PASSWORD = 'MyStore2024!';`
5. Save file

### Step 2: Upload to GitHub
1. Go to your repository
2. Click "Add file" → "Upload files"
3. Drag and drop ALL files from outputs folder
4. Files will upload together
5. Click "Commit changes"
6. Wait 2 minutes for deployment

### Step 3: Test
1. Go to: https://YOUR-USERNAME.github.io/YOUR-REPO-NAME
2. Website should load (not blank!)
3. Try searching for products
4. Try adding to cart
5. If blank → Read FIX_BLANK_PAGE.md
